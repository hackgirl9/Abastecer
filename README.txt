-- App    // Contiene los archivos de la aplicación.
   ---- Assets    // Contiene los archivos de diseño de la aplicación.
   ---- Config    // Contiene los datos de configuración global de la aplicación.
   ---- Controller   // Contiene los controladores de la aplicación.
   ---- Core   // Contiene manejadores, helpers, entidades y controladores principales de la aplicacíon.
   ---- Database // Contiene los datos de la configuración de la base de datos y el Script de la base de datos.
   ---- Libs   // Contiene librerias independientes al proyecto o creadas por los desarroladores.
   ---- Model   // Contiene los modelos (clases) de la base de datos de la aplicación.
   ---- View   // Contiene todas las vistas de la aplicación.
	--- Public   // Contiene los archivos publicos de las vistas.

-- Vendor   // Carpeta que se encarga de la autocarga de clases con Composer. 

-- index.php // Archivo que se encarga del inicio de la aplicación.
